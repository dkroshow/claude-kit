# File-Knowledge Map

> Maps source files to the learnings and docs that cover them.
> Maintained by `/_wrapup`. Used by `/_wrapup` staleness check.
>
> **How it works:** When `/_wrapup` creates a learning with a `## Key Files` section,
> it adds entries here mapping those files to the learning. During the next wrapup,
> changed files are cross-referenced against this index to detect stale knowledge.

> A single learning may have multiple rows (one per key file).
